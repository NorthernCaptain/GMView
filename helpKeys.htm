Knowhere Hot Keys

Track save and load
  • F1 - this help
  • F2 - save recording track into a file
  • F3 - load track from file
 Information panels
  • F4 - show GPS information panel
  • F5 - show Track information panel
  • F6 - show Zoom panel
Recording track
  • F7 - change Map type (Google, Yandex)
  • F8 - turn ON/OFF recording track
  • F9 - show recording track on the map
 Window size
  • F11 - full screen mode ON/OFF
  • F12 - minimize window
