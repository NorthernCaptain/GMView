Knowhere Hot Keys

Hot keys:

  • F1 - this help
  • F2 - save recording track into a file
  • F3 - load track from file
Information panels
  • F4 - show GPS information panel
  • F5 - show Track information panel
  • F6 - show Zoom panel
Recording track
  • F7 - change Map type (Google, Yandex, Openstreet...)
  • F8 - turn ON/OFF recording track
  • F9 - show recording track on the map or not
