Knowhere Hot Keys

Track save and load
  • F1 - this help
  • F2 - save recording track into a file
  • F3 - load track from file
 Information panels
  • F4 - show GPS information panel
  • F5 - show Track information panel
  • F6 - show Zoom panel
 Map navigation
  • WSAD - move map up, down, left, right
  • E,Q - rotate map clockwise or cc
  • PgUp/+,PgDn/- - zoom map in or out
Recording track
  • F7 - change Map type (Google, Yandex)
  • F8 - turn ON/OFF recording track
  • F9 - show recording track on the map
 Window size
  • F11 - full screen mode ON/OFF
  • F12 - minimize window
 GPS related
  • Y - auto rotate map by GPS azimuth
  • N - mark point of the track by GPS position
  • Space - map follows GPS position (autocenter)
Display options
  • Tab,~ - show/hide mini map
  • P - show/hide traffic, if available
  • O - show blocks presence on next zoom lvl
 Quick menu access
  • B - show POI menu
  • T - show Track menu
  • V - show View menu
 Other
  • M - toggle night view mode
